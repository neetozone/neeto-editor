### **Custom keyboard shortcuts**

By default the v2 editor submits the content on `⌘ + Enter` (Mac) or
`Ctrl + Enter` (Windows). An `onSubmit` prop can be provided to call a function
when the content is submitted. It accepts the resulting HTML content as the
argument.

Additional keyboard shortcuts can be registered via the `keyboardShortcuts`
prop. The key is the key combination, and the value is the function to call
when it fires. Use `Mod` as a shorthand for `Cmd` on Mac and `Control`
elsewhere. Return `true` from the handler to prevent the default behavior.
Keyboard shortcuts are only supported in rich-text mode.

<Canvas>
  <Editor
    onSubmit={content => console.log(content)}
    keyboardShortcuts={{
      "Shift-Enter": ({ editor }) => {
        alert(editor.getHTML());
        return true;
      },
    }}
  />
</Canvas>
