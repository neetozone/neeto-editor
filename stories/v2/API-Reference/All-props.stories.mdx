# **FormikEditor props**

#### **shouldFormikEditorUpdate**

The `shouldUpdate` prop in `FormikEditor` decides whether the field should
re-render when its props change. It receives a function called with two
arguments:

- `nextProps`: the next set of props.
- `prevProps`: the previous set of props.

Return `true` to trigger a re-render. Use this to compare props that should
trigger an update, such as `variables` or `mentions`.

**Example:**

```javascript
<FormikEditor
  shouldUpdate={(nextProps, prevProps) => {
    return nextProps.variables !== prevProps.variables || nextProps.mentions !== prevProps.mentions;
  }}
/>
```

As of now, `shouldUpdate` only considers changes in the `mentions` and
`variables` props.

#### **Note**

> The empty value or initialValue should always be `<p></p>`.
