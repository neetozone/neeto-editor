# Error state — v2

neetoEditor can be configured with a custom `error` prop, which renders inside
the editor wrapper. The error prop can be a string, an array of strings, or an
object with a `message` property.

<Canvas>
  <Editor
    error={{
      message: "Something went wrong in the editor",
    }}
  />
</Canvas>
