# Editor height — v2

neetoEditor can be configured to provide a custom initial height using the
`rows` prop. It accepts an integer value which represents the number of rows.
The `heightStrategy` prop controls whether the height is `fixed` or
`flexible`.

<Canvas>
  <Editor rows={3} heightStrategy="flexible" />
</Canvas>
