# Variable Support — v2

neetoEditor supports variable placement. Pass an array of variables as the
`variables` prop. Users can choose from the available variables via the `{}`
icon in the fixed menu, or by typing `{{` inside the editor.

### **Variables prop**

```js
[
  {
    category: "ticket",
    label: "Ticket",
    variables: [
      {
        label: "ID",
        key: "id",
        value: "001",
      },
      {
        label: "Name",
        key: "name",
        value: "Ticket name",
      },
    ],
  },
  {
    label: "Subdomain",
    key: "subdomain",
    value: "subdomain name",
  },
  {
    label: "User",
    key: "user",
  },
  {
    category: "organisation",
    label: "Organisation",
    variables: [
      {
        label: "ID",
        key: "id",
      },
      {
        label: "Name",
        key: "name",
        value: "Organisation name",
      },
    ],
  },
];
```

### **substituteVariables** method

The `substituteVariables` method substitutes variables in the editor content
with their associated values.

```jsx
const parsedEditorContent = substituteVariables(editorContent, variables);
```

Here `variables` is an array of the format shown above.
