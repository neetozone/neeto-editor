# Override defaults — v2

By default the following options are available:

- `bold`
- `italics`
- `underline`
- `strike`
- `link`
- `paragraph`
- `h1`
- `bulleted-list`
- `numbered-list`

You can edit the available options by passing an array of strings to the
`defaults` prop. In addition to the above options you can also pass: `h3`,
`h4`, `h5`, and `h6`.

<Canvas>
  <Editor defaults={["p", "h1", "h2", "h3", "h4", "h5", "h6", "bold"]} />
</Canvas>
