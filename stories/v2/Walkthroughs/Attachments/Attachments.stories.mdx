# Attachments — v2

`import { Attachments } from "@bigbinary/neeto-editor/v2";`

The v2 `Attachments` component allows users to upload files via drag-and-drop
or a file input. The internal upload pipeline still uses `Uppy`, while the UI
shell is rebuilt on neeto-atoms `Button`, `Toastr`, `DropdownMenu`, `Input`,
`Tooltip`, `Spinner`, `Alert`, and `Dialog`.

### **Default**

The attachments prop is an array of objects with the following structure:

```js
[
  {
    filename: "example.jpg",
    url: "https://example.com/images/example.jpg",
    signedId: "ASDF1234",
  },
];
```
