# Custom upload trigger — v2

The upload action of the v2 `Attachments` component can be wired to any
trigger. Forward a ref and call `handleUploadAttachments` on it.

<Canvas>
  <CustomAttachmentsDemo />
</Canvas>

## Usage

```jsx
import React, { useRef } from "react";

import { Button } from "@bigbinary/neeto-atoms";
import { noop } from "neetocist";

import { Attachments } from "@bigbinary/neeto-editor/v2";

const CustomAttachmentsDemo = () => {
  const attachmentsRef = useRef(null);

  const handleUpload = () =>
    attachmentsRef.current?.handleUploadAttachments() || noop;

  return (
    <>
      <Button onClick={handleUpload}>Custom upload button</Button>
      <Attachments
        className="pt-2"
        isIndependent={false}
        ref={attachmentsRef}
      />
    </>
  );
};

export default CustomAttachmentsDemo;
```
