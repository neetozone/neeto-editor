# Custom drop zone for attachments — v2

The drop zone can be specified by providing a ref (`dragDropRef`) to the
wrapper component and passing it as a prop to the `Attachments` component. In
the below example, the entire `Sheet` body is a drop zone.

<Canvas>
  <CustomDragDropDemo />
</Canvas>

## Usage

```jsx
import React, { useState, useRef } from "react";

import { Button, Sheet } from "@bigbinary/neeto-atoms";

import { Attachments } from "@bigbinary/neeto-editor/v2";

const CustomDragDropDemo = () => {
  const [showSheet, setShowSheet] = useState(false);
  const [attachments, setAttachments] = useState([]);

  const dragDropRef = useRef(null);

  return (
    <>
      <Button label="Show Sheet" onClick={() => setShowSheet(true)} />
      <Sheet isOpen={showSheet} onClose={() => setShowSheet(false)}>
        <Sheet.Header>
          <Sheet.Title>Custom drop zone</Sheet.Title>
        </Sheet.Header>
        <Sheet.Body>
          <div
            className="flex w-full h-96 ne-attachments__wrapper"
            ref={dragDropRef}
          >
            <Attachments
              attachments={attachments}
              dragDropRef={dragDropRef}
              onChange={setAttachments}
            />
          </div>
        </Sheet.Body>
      </Sheet>
    </>
  );
};
```
