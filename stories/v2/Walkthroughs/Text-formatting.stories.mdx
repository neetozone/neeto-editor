# Formatting text in the rich text editor — v2

<br />

neetoEditor offers a diverse range of features beyond basic text formatting. In
addition to bold, italic, inline code, and strikethrough, you can integrate
links, embedded content, images, videos, and other attachments. The v2 editor
ships the same feature surface, rebuilt on top of `@bigbinary/neeto-atoms`.

[Learn more about formatting text](https://help.neetocal.com/articles/formatting-text-in-rich-text-editors).
